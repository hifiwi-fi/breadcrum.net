Apple Shortcuts let you save Safari pages to Breadcrum from the share sheet. Eventually this will be provided by a native app.

- [Breadcrum iOS shortcut](https://www.icloud.com/shortcuts/2067714b7f974298886570256528bf46): Create a bookmark and show the edit prompt.
- [Breadcrum instant iOS shortcut](https://www.icloud.com/shortcuts/73f889f9d0d544488e247bc522e524bb): Create a bookmark without any prompts. Requires an [auth token](/docs/account/auth-tokens/).
