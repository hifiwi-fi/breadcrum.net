Stars are a lightweight way to mark bookmarks you want to surface. Starred bookmarks can be filtered across bookmarks, archives, and episodes views.

You can toggle the starred filter from the top navbar (star icon) on any signed-in page.

Stars are a bookmark flag. Toggling a star updates the bookmark immediately and affects related archive and episode lists when you enable the starred filter.
